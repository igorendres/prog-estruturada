Conceitos Fundamentais em C para este Exercício
Para construir este programa, vamos precisar dos seguintes conceitos da linguagem C:

Funções (functions): Blocos de código que realizam uma tarefa específica e podem ser chamados de outras partes do programa. Isso ajuda a organizar e reutilizar o código.

Variáveis (variables): Espaços na memória para armazenar dados. Elas possuem um tipo (como int para números inteiros) e um nome.

Tipos de Dados (data types): Define o tipo de dado que uma variável pode armazenar. Neste caso, usaremos o int para números inteiros.

Entrada e Saída Padrão (stdin e stdout): Funções para interagir com o usuário.

printf(): Para "imprimir" ou exibir informações na tela.

scanf(): Para "escanear" ou ler dados digitados pelo usuário no teclado.

Estruturas Condicionais (if-else): Permitem que o programa tome decisões e execute diferentes blocos de código com base em certas condições (por exemplo, se um número é maior que outro).

Sintaxe Básica: A "gramática" da linguagem, incluindo o uso de ponto e vírgula (;) no final das instruções, chaves ({}) para delimitar blocos de código e a função main como ponto de partida do programa.

Inclusão de Bibliotecas (#include): Como importar bibliotecas que contêm as funções que precisamos, como a stdio.h para entrada e saída.
